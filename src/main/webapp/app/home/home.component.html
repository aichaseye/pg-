Bienvenue sur la Plateforme de Gestion des Matricules (PGM)

Ministere de la Formation de l'Apprentissage et de l'Insertion

BON A SAVOIR

PGM est un générateur automatique de Numéro d'identifiant unique (NIU). Il permet d'immatriculer de façon unique : Les établissements (CFP et LT), les apprenants des EFPT, Les enseignants (vacataires et contractuels), Les matières du Ministère afin de préconiser la transparence la qualité et dans le systéme la formation.

NIU pour les apprenants

le personnel administratif de l'établissement crée les matricules
l'apprenant doit créer un compte pour visualiser son numéro
ou pouvoire effectuer une demande

NIU pour les Enseignants

le Ministere crée les matricules
l'enseignant doit créer un compte pour visualiser son numéro
ou pouvoir effectuer une demande

Codification pour les Etablissements

le BFPA crée les identifiants
le personnel administratif crée un compte
il poura saisir une demande

Codification pour les Matiéres

Pour les matériels du ministére
le ministére immatricule mes matières avant de les faire sortire du ministére
